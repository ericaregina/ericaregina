### Sobre mim

Profissional de BI com atuação em modelagem de dados, desenvolvimento de dashboards e relatórios de gestão. Tenho experiência prática construindo indicadores de SaaS (GMV, TPV, MRR, churn) e análises de ciclo de vida de clientes em ambiente de produto real, utilizando Power BI, SQL (incluindo ClickHouse) e Apache Superset.

Trabalho todo o fluxo de dados: modelagem (star schema), DAX, Power Query, escrita de queries SQL para extração e transformação, e construção de visuais customizados (HTML/SVG) quando o Power BI nativo não é suficiente. Também uso Python para automação de análises, geração de gráficos e rotinas de limpeza de dados.

Tenho formação complementar em desenvolvimento web e versionamento com Git, o que me ajuda a entender a ponta de dados de aplicações reais e a colaborar melhor com times de engenharia.
